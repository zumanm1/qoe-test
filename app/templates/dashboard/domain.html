{% if domain == 'ran' %}
RAN Domain Information

The RAN (Radio Access Network) domain is responsible for wireless connectivity to user equipment:

  • UE (User Equipment): Mobile devices that connect to the network (phones, modems, IoT devices)
  • eNodeB: Base stations that provide cellular radio coverage and connectivity to core network

Note: The RAN domain forms the foundation of the mobile network and is responsible for efficient radio resource management.

{% elif domain == 'transport' %}
Transport Domain Information

The TRANSPORT domain consists of three IP-MPLS domains:

  • IP-MPLS Backhaul: Connects RAN side via IP VRF to the Core Data Network using Inter-AS Option A. Uses microwave (10Mbps-2.5Gbps) and fiber (100Mbps-100Gbps) mediums. Router naming convention: PEx1--Px0, PExx2
  • IP-MPLS Core Data Network: Provides connectivity between Core Network elements (S-GW, P-GW, IGW). Router naming convention: PEyy1, Pyy0, PEyy2
  • IP-MPLS International/Internet: Provides connectivity to external networks and internet. Router naming convention: PE1, P0, PE2

Transport Technologies: Fiber, Microwave, and DWDM are the primary transmission media used in the transport network.

{% elif domain == 'core' %}
Core Domain Information

The CORE domain consists of key network components that manage mobility, authentication, and user plane traffic:

  • MME (Mobility Management Entity): Controls signaling for mobility and security
  • HSS (Home Subscriber Server): Central user database for authentication and subscription information
  • PCRF (Policy and Charging Rules Function): Policy enforcement and charging control
  • S-GW (Serving Gateway): User plane gateway that routes and forwards data packets
  • P-GW (PDN Gateway): Connects to external packet data networks and provides IP address allocation
  • IGW (Internet Gateway): Gateway to the internet

Note: The Core Domain handles all centralized processing functions in the mobile network.

{% elif domain == 'internet' %}
Internet Domain Information

The INTERNET domain provides connectivity to external networks and content providers:

  • PE1 (Provider Edge 1): Edge router connecting to international networks
  • P0 (Provider Core): Core router providing transit services
  • PE2 (Provider Edge 2): Edge router connecting to content providers
  • Ookla Server: Speed test server for network performance testing

Note: The Internet Domain enables mobile subscribers to access content and services on the global internet.
